System Architecture & Data Flow

How It Works

1

Camera captures real-time video frames

2

YOLOv8 model processes and detects objects

3

Detected labels sent to Flask API backend

4

gTTS engine converts text to multilingual audio

5

Audio output delivered to user via bone conduction

Database Logging

The system logs all interactions (detected objects, selected language, timestamps) to SQLite database via Flask backend for analytics and improvement.

Technology Stack

  • • YOLOv8 for object detection
  • • Python & OpenCV for processing
  • • Flask API backend
  • • SQLite database
  • • gTTS multilingual engine
